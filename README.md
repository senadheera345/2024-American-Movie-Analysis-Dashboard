2024 Movie Analysis Dashboard (Power BI)

An end-to-end data project that transforms raw movie data into an interactive and visually engaging Power BI dashboard. This project focuses on analyzing 2024 American movies to uncover trends in genres, production companies, and release patterns.

🚀 Project Overview

This project demonstrates the complete data pipeline:

Data scraping
Data cleaning & transformation
Data modeling
Interactive dashboard design
📥 Data Collection
Scraped movie data using Beautiful Soup
Enriched dataset with additional details (IMDb ratings, genres, release dates) via the OMDb API
API Link: https://www.omdbapi.com/
🛠️ Data Preparation

Performed data cleaning and transformation using Power Query:

Converted runtime into minutes
Created custom categories (e.g., weekday vs weekend releases)
Added calculated columns for deeper analysis
📊 Dashboard Features

The Power BI dashboard provides insights such as:

🎬 Most popular genres and production companies
⭐ IMDb ratings by production company
📆 Movie releases by month
🎛️ Dynamic slicers for filtering (genre, weekday/weekend)
🎨 Design Highlights
Dark-themed cinematic UI (#2C2C2C)
KPI indicators using gold and red accents
Dynamic movie poster visuals linked via URLs
💡 Key Learnings
Integrated web scraping with API data enrichment
Applied data transformation techniques in Power Query
Built interactive and user-friendly dashboards
Enhanced storytelling through data visualization
